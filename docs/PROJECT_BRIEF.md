# CRM VP — Project Brief

## Objetivo

Crear un CRM interno para Soluciones VP que permita:

- Registrar prospectos y clientes.
- Dar seguimiento comercial ordenado.
- Usar pipeline tipo Kanban.
- Automatizar recordatorios y mensajes.
- Conectar posteriormente con WhatsApp y n8n.

## Stack

- Backend: Python + FastAPI
- Base de datos: PostgreSQL
- ORM: SQLAlchemy
- Migraciones: Alembic
- Frontend: React + Vite
- Infraestructura: Docker
- Deploy futuro: Railway

## Fase actual

Fase 1: creación manual del proyecto y estructura base.

## Regla de desarrollo

Avanzar máximo 3 pasos a la vez.
No crear módulos grandes antes de validar la estructura.
No integrar WhatsApp ni n8n hasta tener CRM base estable.
